How to bypass Windows Defender

Red Alert Security is an indie app, so Windows SmartScreen may flag it. This is completely normal for any unsigned application — it does not mean the file is harmful. Just follow two quick steps:

01

Click "More info" on the SmartScreen warning

When you launch the installer, Windows will show a blue SmartScreen dialog. Do not close it. Look for the "More info" link near the bottom-left of the window and click it.
